# ⚡ CopyKilat

AI Copywriter khusus Bahasa Indonesia. Bikin caption Instagram/TikTok, deskripsi produk Shopee/Tokopedia, dan copy iklan dalam hitungan detik.

## Fitur

- 📱 **Caption IG/TikTok** — Caption engaging yang bikin orang stop scrolling
- 🛒 **Deskripsi Produk** — Deskripsi Shopee, Tokopedia, TikTok Shop, Lazada
- 📢 **Copy Iklan** — Copy ads untuk Facebook, Google, TikTok Ads

## Tech Stack

- React + Vite
- Gemini API (Google AI)
- Deploy di Vercel (gratis)

## Setup

```bash
npm install
npm run dev
```

## Cara Pakai

1. Ambil Gemini API key gratis di [aistudio.google.com/apikey](https://aistudio.google.com/apikey)
2. Buka app → klik ⚙️ Set API Key → paste key
3. Pilih mode → isi form → Generate!
